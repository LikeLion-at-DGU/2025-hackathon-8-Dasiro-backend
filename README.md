### 💻 사용 기술 스택

| 분야 | 기술 |
|------|------|
| **언어 & 프레임워크** | Python, Django, Django REST Framework (DRF) |
| **서버 & 배포** | AWS EC2 (Ubuntu), Gunicorn, Nginx |
| **CI/CD** | GitHub Actions - 자동 배포 + 무중단 배포 |
| **DB** | SQLite3 (개발용), PostgreSQL (전환 고려 중) |
| **기타** | Docker (선택), Map API (지도 서비스) |
